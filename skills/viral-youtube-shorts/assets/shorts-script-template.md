# Shorts script template

A 30 to 45 second YouTube Shorts skeleton. Fill in every slot. Keep the
audio column tight (spoken word lands faster than read text), keep the
on-screen text in the safe zone (middle third of the frame), and design
the last beat to loop back into the first.

```
WORKING TITLE:           (search-friendly if you want long-tail; concrete either way)
HOOK ARCHETYPE:          (curiosity gap / withhold / stake / contrarian / direct call-out / number / in media res)
TARGET LENGTH:           (30 to 45s recommended; 15s+ for watch-time minimum)
GOAL:                    (Shorts Feed reach / long-form click / search / monetization)
LONG-FORM TO BRIDGE TO:  (exact title and chapter, or "none, write the long-form first")

──────────────────────────────────────────────────────────────────────
TIME    | SPOKEN AUDIO                     | ON-SCREEN TEXT (safe zone) | VISUAL / SHOT
──────────────────────────────────────────────────────────────────────
0 to 3s | HOOK LINE                        | HOOK, 3 to 5 words, big    | first frame is the hook
        | (lead with payoff or stake;      |                            | (no logo, no intro,
        |  land inside 1.5 seconds)        |                            |  no welcome-back)
        |                                  |                            |
3 to 8s | ESCALATION beat 1                | key phrase                 | reset shot or angle
        | (turn, stakes raise, or          |                            |
        |  specificity injection)          |                            |
        |                                  |                            |
8 to    | ESCALATION beat 2                | key phrase                 | reset shot or angle
~25s    | (use "but / therefore" not       |                            |
        |  "and then")                     |                            |
        |                                  |                            |
~25 to  | PAYOFF                           | the takeaway               | show the result
~38s    | (deliver the hook's promise)     |                            |
        |                                  |                            |
last 5s | LONG-FORM CALLOUT                | label + arrow              | hold on result OR
        | (name the long-form by topic,    |                            | match opening frame
        |  not "subscribe")                |                            | for the loop
──────────────────────────────────────────────────────────────────────

LOOP DESIGN:             (visual loop / audio loop / question loop / cliff loop)
RE-HOOK LINE (optional): (a "but here's the part nobody expects" beat around 8 to 10s)
ORIGINAL AUDIO?:         (yes / no; if licensed, accept the revenue split)
ASPECT RATIO:            (9:16 vertical; not 16:9 with bars)
CAPTIONS BURNED IN?:     (yes; in safe zone)
PROMISE / PAYOFF MATCH:  (does the body actually deliver what the hook promised?) [ ]
```

Notes:

- If the hook needs a sentence of setup, the setup is your real hook.
  Start there.
- Cut the first and last lines last. They're almost always over-written.
- A flat retention curve above the gray band beats a curve that spikes
  high then crashes. Aim for steady, not dramatic.
- If you can't write a long-form callout that names a real long-form,
  consider whether the long-form should exist first. See
  `references/shorts-to-longform-funnel.md`.
