# Pinned comment template

Fill one of these in per video. The pin is a second surface most
creators leave empty. See `references/pinned-comments.md` for the
reasoning and the per-platform reality.

## Pick one pattern

### A. Objection-killer

Take the predictable critical comment off the board.

```
Yes, [the obvious objection]. I tested [the specific case] and
[the honest result].
```

Example skeletons:

- "Yes, this works on oily skin too. I tested it for 14 days."
- "Before you say 'sponsored': I bought all three myself."
- "I've had three people ask if it works on [X]. Yes, same routine,
  same result."

### B. Second hook / context extension

A line that answers the obvious next question and keeps the viewer in
the post.

```
[The follow-up the viewer was about to ask], answered: [one-line
answer that opens a new small loop].
```

Example skeletons:

- "Day 14 update: still no flaking, no drop in strength."
- "Yes, the cheaper one beat the $800 one on shot consistency too."
- "The one I didn't show on camera is the one that actually fixed it."

### C. Link / list / what didn't fit

The product list, the link, the recipe steps that couldn't go in the
caption cleanly.

```
[Label]:
- [item 1]
- [item 2]
- [item 3]
[link if any]
```

Example skeletons:

- "Products in order: [brand A, shade 12], [brand B, shade light],
  [brand C, neutral]."
- "Full recipe with the substitutions I tested: [link]."
- "Long-form breakdown is on my channel under 'Why retinol peels'."

### D. Second-hook into a related video

Sends the viewer to another piece of yours. Works best when the
original opens a loop the related video closes.

```
If you liked this, [the related video] is the [prequel / sequel /
deeper version]. It's on my profile.
```

## Per-platform notes (fill in)

```
PLATFORM:       (Instagram / TikTok / YouTube Shorts)
PATTERN:        (A objection-killer / B second hook / C link-list / D related)
PIN MECHANIC:
  - Instagram:    pin from your own account, up to 3, newest-pinned-first
  - TikTok:       reply-with-video to a seeded or real comment (pin not reliable)
  - Shorts:       heart + pin combo from creator account
TIMING:         pin within the first 30–60 minutes of publishing
EDIT LATER:     yes, swap or edit as conversation develops
```

## Don'ts

- Do not put "comment YES for the link" or any bait-shaped trigger in
  the pin. Run it through `bait-check.md`.
- Do not repeat the caption or the on-screen hook in the pin. Wasted
  slot.
- Do not link to a bait-and-switch (free thing in the caption, sales
  page at the link).
- Do not pin a generic "thanks for watching, follow for more". That's
  two anti-patterns in one line.
