# TikTok script template (21–34s)

The default cold-reach shape. Target completion ~70%. Use for hooks,
demos, talking-head, or before-after content. For 60s+ Creator Rewards
scripts, see `long-tiktok-template.md`.

Keep the audio column tight (people talk faster than they read), and
make the on-screen text reinforce the spoken line, not duplicate it.
Cut shot or angle every 2–3 seconds.

```
WORKING TITLE:
FORMAT:            (talking head / demo / before-after / GRWM / stitch / shoppable)
GOAL:              (reach / saves / follows / shop clicks)
TARGET LENGTH:     21–34s
ACCOUNT TYPE:      Creator / Business
SOUND:             trending (sound name + spotted date) / original
LOOP STRATEGY:     (final frame loops to / final line resolves to)

──────────────────────────────────────────────────────────────────────
TIME    | SPOKEN / AUDIO                | ON-SCREEN TEXT      | VISUAL / SHOT
──────────────────────────────────────────────────────────────────────
0–3s    | HOOK (payoff or stake first) | hook in 3–5 words   | in media res / face / product close-up
        |                              |                     |
3–8s    | ESCALATION 1                 | key phrase          | reset shot/angle
        | (add the specific number     |                     |
        |  or named thing)             |                     |
        |                              |                     |
8–18s   | ESCALATION 2                 |                     | reset shot/angle
        | (raise stakes or reveal      |                     |
        |  context. use "but" not      |                     |
        |  "and then")                 |                     |
        | [RE-HOOK ~s12 if energy sags]|                     |
        |                              |                     |
18–28s  | PAYOFF (visible result)      | the takeaway        | show, do not say
        |                              |                     |
28–34s  | LOOP or CTA (one ask, light) | CTA text            | final frame matches opening
──────────────────────────────────────────────────────────────────────

OPEN LOOP (set in hook → closed at payoff):
RE-HOOK LINE (for the mid-video sag):
HOOK ARCHETYPE: (middle-of-action / contrarian / confession / withhold / visual+verbal / sensory)
PROMISE↔PAYOFF CHECK:  does the body deliver what the hook promised?  [ ]
3-SECOND CLIFF CHECK:  hook lands before second 2?  [ ]
LOOP CHECK:            final frame reads as the opening?  [ ]
```

Notes:

- If the hook needs a sentence of setup, the setup is probably the real
  hook. Start there.
- Cut the first and last lines last. They are usually the most
  over-written.
- No dead air at start or end. No logo, no "welcome back," no musical
  sting before the first spoken line.
