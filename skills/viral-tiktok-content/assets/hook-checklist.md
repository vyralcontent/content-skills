# TikTok hook checklist

Run any candidate hook through this. A TikTok hook has 1–3 seconds to
clear the cliff before the algorithm starts reading swipe-away. If a hook
fails the first four checks, rewrite it before going further.

- [ ] **Lands before second 2** when said out loud. No setup, no "hey guys."
- [ ] **Leads with payoff or stake**, not context, brand, or "today I'm
      going to talk about."
- [ ] **First frame has motion or a striking visual.** Not a logo, not a
      static face, not an intro animation.
- [ ] **Works on mute.** On-screen text (3–5 words) carries the hook even
      with sound off.
- [ ] **Works with sound.** The spoken line lands cleanly and matches the
      visual energy.
- [ ] **Concrete.** Has a specific noun, number, or named stake.
- [ ] **Opens a loop or states a consequence** the viewer needs resolved.
- [ ] **Promise is one the script actually pays off** (no orphan
      clickbait).
- [ ] **Targets a specific viewer.** They feel "this is for me" in the
      first second.
- [ ] **Distinct archetype** from the other options in the set (real
      diversity, not variations).
- [ ] **Sound choice matches the hook energy.** Trending audio, original
      voiceover, or diegetic demo sound.

Fast upgrades when a hook is weak:

- Add a specific number or a dollar/time figure ("$39", "day 47", "in 4
  weeks").
- Name the exact viewer ("if you have oily skin and you also …").
- Cut everything before the first interesting word.
- Swap a vague benefit ("level up your routine") for a concrete result
  ("4 weeks, no peeling").
- Turn a statement into a withheld reveal ("I won't say the brand, but…").
- Start in the middle of the action instead of the setup.

TikTok-specific traps to check for:

- [ ] No TikTok watermark if this is cross-posted from another platform.
- [ ] No "follow for more" or "like and share" as the hook.
- [ ] No channel-format opening ("welcome back to my channel").
- [ ] No musical sting before the first spoken line (reads as dead air).
- [ ] If AI-generated showing realistic people, disclosure toggle is on.
