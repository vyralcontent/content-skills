# TikTok Shop video template

Problem-first shoppable video. Use for affiliate posts, owned-seller
content, or any post with a product card attached. Target 21–34s for cold
reach, 45–60s for trust-building demo. Hook on the problem, product in
frame by second 6, real-use demo, soft close.

The shape that converts is the same shape that earns: problem framing
holds the cliff, product visibility drives brand affinity, and the
shoppable card is the close (no hard verbal CTA needed).

```
WORKING TITLE:
PRODUCT:
PRICE:
COMMISSION (if affiliate):     ___% on ___ category
LINK / SHOP CARD:              (TikTok Shop URL or affiliate)
TARGET LENGTH:                 21–34s (cold reach) / 45–60s (demo)
ACCOUNT:                       Creator (Affiliate program) / TikTok Shop seller

──────────────────────────────────────────────────────────────────────
TIME    | SPOKEN / AUDIO                | ON-SCREEN TEXT      | VISUAL / SHOT
──────────────────────────────────────────────────────────────────────
0–3s    | PROBLEM HOOK                 | the problem in 3–5  | the problem, mid-scene
        | (the pain, not the product)  | words               |
        |                              |                     |
3–6s    | PROBLEM ESCALATION           | "I tried 4 things"  | quick montage of failed
        | (specifics: the cost, the    |                     | attempts, frustration
        |  attempts, the duration)     |                     |
        |                              |                     |
6s      | PRODUCT APPEARS              | product name        | product in frame, real use
        | (no pitch yet, just the      |                     | (NOT static product shot)
        |  introduction)               |                     |
        |                              |                     |
6–18s   | REAL-USE DEMO                | key benefit         | watch the product solve
        | (the product solving the     |                     | the problem
        |  problem in real time)       |                     |
        |                              |                     |
18–28s  | PROOF / RESULT               | before-after frame  | visible result, not
        | (visible, not claimed)       |                     | claimed
        |                              |                     |
28–34s  | SOFT CLOSE                   | "tap the cart"      | product on screen, card
        | ("link's in the cart")       |                     | visible
──────────────────────────────────────────────────────────────────────

PROBLEM-FIRST CHECK:    is the hook on the pain, not the product?    [ ]
6-SECOND PRODUCT CHECK: is the product on screen by second 6?         [ ]
DEMO CHECK:             is the product shown solving the problem?     [ ]
SOFT-CLOSE CHECK:       is the verbal CTA light (card does the work)? [ ]
TRUST LAYER:            does the audience already trust this voice?
                        if no, post visibility/trust content first.   [ ]
AD-SHAPED CHECK:        does it feel native, or does it read like     [ ]
                        a deck cross-post?
```

Notes:

- Direct-response openings ("buy this now") underperform because the
  surface reads ad-shaped and the second-3 cliff fires.
- If you do not have the trust layer yet, the shoppable post will
  underperform regardless of how clean the demo is. Build visibility
  and trust content first (founder, behind-the-scenes, real-use without
  the link), then layer in the conversion posts.
- For TikTok Shop sellers: LIVE shopping highlights can auto-clip into
  shoppable videos. Mark the strong demo moments in your LIVE stream
  deliberately.

Honest framing for the client:

- Quote in saves and shop clicks, not just views.
- Set expectation: a strong product post can take 48–72 hours to break
  out of the seed cohort.
- Never promise FYP placement. Promise the format and the shape.
