# Long TikTok script template (60s+)

Use this for Creator Rewards eligibility (videos must be over 60 seconds
to earn) or for story-driven content that needs the runway. Target
completion drops to ~50% at 60s and ~40% at 90s+. Two retention resets
are mandatory.

If the story is honestly 30 seconds, do not stretch. Pad-stretching
loses on completion rate and earns nothing.

```
WORKING TITLE:
FORMAT:            (deep-dive demo / story / multi-step tutorial / case study)
GOAL:              (Creator Rewards earnings / saves / authority build)
TARGET LENGTH:     60s / 90s / 120s
ACCOUNT TYPE:      Creator (Business is not eligible for Rewards)
SOUND:             original (recommended for long-form) / trending
PAYOFF STACK:      payoff 1 = ___ , payoff 2 = ___

──────────────────────────────────────────────────────────────────────
TIME     | SPOKEN / AUDIO                | ON-SCREEN TEXT     | VISUAL / SHOT
──────────────────────────────────────────────────────────────────────
0–3s     | HOOK (no softening, same     | hook in 3–5 words  | in media res / striking visual
         |  rules as the short shape)   |                    |
         |                              |                    |
3–10s    | PROMISE STACK                | "what you'll see"  | quick montage / preview frames
         | ("here's what you'll see")   |                    |
         |                              |                    |
10–25s   | ESCALATION 1                 | key phrase         | reset shot/angle
         | (open the first loop)        |                    |
         |                              |                    |
25–28s   | RE-HOOK 1                    | "but here's…"      | visual reset, new angle
         | ("but here's the part        |                    |
         |  nobody expects")            |                    |
         |                              |                    |
28–45s   | ESCALATION 2                 |                    | reset shot/angle
         | (close first loop, open      |                    |
         |  the next)                   |                    |
         |                              |                    |
45–52s   | PAYOFF 1 (visible result)    | takeaway 1         | show, do not say
         |                              |                    |
52–60s   | LOOP / CTA                   | final text         | final frame loops to opening
         | (for 60s exactly: end here)  |                    |
──────────────────────────────────────────────────────────────────────
[For 90s+ continue:]
──────────────────────────────────────────────────────────────────────
60–65s   | RE-HOOK 2                    | "and then this…"   | visual reset, change pace
65–85s   | ACT 2 (second substantive    |                    | reset shot/angle
         |  beat, build to payoff 2)    |                    |
85–95s   | PAYOFF 2 (the implication    | takeaway 2         | show, do not say
         |  or the second result)       |                    |
95s+     | LOOP / CTA                   | CTA text           | final frame loops to opening
──────────────────────────────────────────────────────────────────────

3-SECOND CLIFF CHECK:    hook lands before second 2?         [ ]
RE-HOOK CHECK (60s):     re-hook at ~s25, visual reset?      [ ]
RE-HOOK CHECK (90s+):    two re-hooks, ~s25 and ~s60?        [ ]
PAYOFF VISIBILITY:       are results shown, not just stated? [ ]
LENGTH HONESTY:          is this honestly 60s+ of substance? [ ]
CREATOR ACCOUNT:         Business accounts earn nothing.     [ ]
```

Notes:

- The re-hooks are the difference between people finishing and people
  bailing at the sag. Do not skip them.
- Long-form on TikTok is hardest at seconds 25–35 and 60–70. Plan a
  visual reset at each.
- Cut shot or angle every 2–3 seconds throughout. The eye should never
  settle into "I have seen enough."
