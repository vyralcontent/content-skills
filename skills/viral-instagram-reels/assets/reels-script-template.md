# Reels script template

Fill this in for a single Reel. Aim for 15 to 30 seconds unless the brief
says otherwise. Total seconds watched matters, so a longer Reel with strong
retention can beat a shorter one with weak retention, but the cold-reach
sweet spot stays short.

Keep the audio column tight. People talk faster than they read, and the
on-screen text should reinforce the spoken line, not duplicate it.

```
WORKING TITLE:
TOPIC / KEYWORD PHRASE:    (the exact words a stranger would search)
NAMED VIEWER:              (the one specific person this Reel is for)
GOAL:                      (cold reach / saves / follows / sends)
TARGET LENGTH:             (sec)
ACCOUNT TYPE:              (Creator / Business)
TRIAL OR PUBLISH:          (see assets/trial-reels-decision.md)

──────────────────────────────────────────────────────────────────────
TIME  | SPOKEN / AUDIO                  | ON-SCREEN TEXT      | VISUAL / SHOT
──────────────────────────────────────────────────────────────────────
0-3s  | HOOK (concrete, named viewer,   | hook headline,      | face / reveal /
      |  payoff promise)                | 3-5 bold words      | in media res
      |                                 |                     |
3-Xs  | ESCALATION beat 1               | key phrase          | reset shot/angle
      |  (raise tension, add specifics) |                     |
      | ESCALATION beat 2               |                     | reset shot/angle
      |   use "but / therefore"         |                     |
      |                                 |                     |
X-Ys  | PAYOFF (deliver the promise,    | the takeaway,       | show the result
      |  named container if possible)   | nameable phrase     |
      |                                 |                     |
end   | SEND-PROMPT CTA                 | CTA text, own       | hold / loop
      | "send this to the friend who…"  | beat                | to start
──────────────────────────────────────────────────────────────────────

CAPTION:
  Line 1: [keyword phrase as headline]
  Body:   [1-2 lines expanding the hook, plain language]
  CTA:    [named-persona send-prompt, restate the one from the Reel]
  Tags:   [3-5 highly relevant hashtags]

AUDIO:
  Source:                     (original / Sound Collection / trending)
  Trending window check:      (upward arrow, < ~30K uses)  [ ]
  Audio attached on test view: [ ]

PROMISE -> PAYOFF CHECK:
  Does the body deliver what the hook promised?  [ ]
  Is the payoff useful to one specific person?   [ ]
  Could that person be named in the CTA?         [ ]

ORIGINALITY PASS (see references/originality-and-audio.md):
  No TikTok watermark visible:    [ ]
  No CapCut watermark visible:    [ ]
  No third-party app UI on screen:[ ]
  Substantial edit (your cuts,    [ ]
   your voice, your text):
```

Notes:
- If the hook needs a sentence of setup, the setup is probably the real
  hook. Start there.
- Cut the first and last lines last. They tend to be the most over-written.
- On Reels, the grid crop is 3:4. If the Reel is going to the grid, frame
  the hook subject in the upper two-thirds so the crop keeps it.
