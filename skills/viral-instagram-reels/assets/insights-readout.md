# Reels Insights readout

Fill this in after a Reel has been live long enough to read (24 hours
minimum for a trial, 72 hours minimum for a public Reel). The point is
to map each metric to the lever it points to, so the next Reel changes
in the right direction.

```
REEL TITLE / TOPIC:
PUBLISHED:                  (date / trial or public)
HOURS LIVE:
ACCOUNT BASELINE NOTES:     (your usual numbers for this kind of Reel)
```

## Numbers

```
Reach:
Total views:
Skip rate:                  (your baseline: __ )
Average watch time:
Retention curve cliff:      (timestamp where drop-off happens)
Share rate (sends/reach):   (your baseline: __ )
Save rate:                  (your baseline: __ )
Likes per reach:            (your baseline: __ )
Comments:
Followers from this post:   (your baseline: __ )
Post-watch: profile visits:
Post-watch: auto-play into next Reel:
```

## Read the numbers (check the one that fits)

- [ ] **High skip rate, low total views.** Hook failed the three-second
      cliff. Fix the opening three seconds before doing anything else.
      (See `references/reels-hook.md`.)
- [ ] **Hook held, retention cliff in the middle.** Body failed. Add a
      re-hook beat or compress the build-up.
- [ ] **High reach and watch time, near-zero follows.** Topic legibility
      failed. The Reel was watchable but didn't make the topic clear.
      Fix the on-screen text and caption keyword.
      (See `references/caption-and-search.md`.)
- [ ] **Watch time fine, sends below baseline.** CTA failed. Recut the
      end with a named-persona send-prompt.
      (See `references/sends-playbook.md`.)
- [ ] **Multiple recent Reels with sudden low reach.** Possible
      originality cap. Audit the last 30 days.
      (See `references/originality-and-audio.md`.)
- [ ] **Audio missing on test view.** Audio was stripped at publish.
      Recut with safer audio.
- [ ] **All metrics healthy, the Reel just hit.** Note the format. Plan
      a follow-up in the same shape.
- [ ] **All metrics flat with no clear failure.** Likely variance or
      topic-fit mismatch. Don't change the format on one outlier.

## Action for the next Reel

```
EARLIEST FAILURE FOUND:
ONE CONCRETE FIX:           (recut opening / new caption / re-trial /
                             fresh Reel built around named-persona payoff)
WHAT TO KEEP:               (what worked, don't change it)
RE-TRIAL OR FRESH?:
```

## Don't read these as verdicts

- View count alone. Without reach and share context, it's vanity.
- Likes alone. Cheapest action, weakest distribution signal.
- Day-one numbers on a public Reel. Reels can keep distributing for
  days if share rate stays healthy.
