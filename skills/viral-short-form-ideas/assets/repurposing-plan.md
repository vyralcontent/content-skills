# Repurposing plan

One trunk asset, many derived branches. Fill this in for each trunk
asset you produce. The goal is to push past the first 5 obvious
derivatives. The leverage is in the next 15.

```

FUNNEL CHECK:
  [ ] Every branch references or links back to the trunk asset.
  [ ] Each short stands alone (a viewer who never sees the trunk gets value).
  [ ] The trunk has a clear next step (subscribe, download, opt-in, buy).
```

Cadence guidance: 1 trunk asset per week or per month, depending on
ambition. Weekly is for full-time creators. Monthly is more common.
Either way, the branches should ship over 3 to 6 weeks following the
trunk, not all in one week.
