# Pillar worksheet

Fill this in once. Keep it visible while you ideate. Revisit every 90
days. Pillars are the constraint every other ideation engine in this
skill builds on, so it pays to get them right before generating a
single idea.

```

SANITY CHECKS:

  [ ] Each pillar answers a question the AUDIENCE has, not just one
      I find interesting.
  [ ] Each pillar can produce 20+ angles without reaching.
  [ ] No two pillars produce the same 10 ideas (if they do, merge).
  [ ] I can describe the niche (top of page) in one sentence after
      filling this in.
```

Revisit cue: re-run this worksheet if idea drought hits for more than
two weeks, or if audience signal (comments, DMs, analytics) is
pointing at an interest none of your current pillars cover.
