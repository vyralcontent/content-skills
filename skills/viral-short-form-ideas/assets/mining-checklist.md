# Mining checklist (weekly)

30 to 45 minutes a week. Set a timer per section so you don't get
sucked into rabbit holes. Drop every raw capture into your idea inbox
without judgement; triage in a separate session.

```
DATE:
TIME BUDGET:  ~30 to 45 minutes
```

## 1. Comment mining (10 min)

- [ ] Open the last 3 posts that beat your account average.
- [ ] Sort comments by top.
- [ ] Pull every question, objection, or "wait but what about…" follow-up.
- [ ] Pull every comment that uses unexpected language for the topic.

```
CAPTURES:
  -
  -
  -
  -
```

## 2. DM mining (5 min)

- [ ] Scan DMs from the last 7 days.
- [ ] Pull every "can you explain X?" or recurring question.

```
CAPTURES:
  -
  -
```

## 3. Reddit (10 min)

- [ ] Pick 2 audience subreddits (audience, not topic).
- [ ] Sort each by "top, all time."
- [ ] Read top 30 post titles; open the 5 closest to your pillars.
- [ ] Read the top comments on those 5, not the post bodies.

```
SUBREDDIT 1:
  CAPTURES:
    -
    -
SUBREDDIT 2:
  CAPTURES:
    -
    -
```

## 4. Autocomplete (5 min)

- [ ] Pick one seed phrase tied to a pillar.
- [ ] YouTube autocomplete: seed + space + a, then b, c, etc.
- [ ] Note the 5 strongest completions.
- [ ] Cross-check the top 1 to 2 in Google's "People Also Ask."

```
SEED:
CAPTURES:
  -
  -
  -
  -
  -
```

## 5. Competitor outliers (10 min)

- [ ] Pick 3 competitors at your size or one tier up.
- [ ] Find one outlier each (3 to 4x channel baseline).
- [ ] For each: name the MECHANIC (comparison, transformation,
      deadline, cost confession, etc.), not the surface.

```
COMPETITOR 1:
  OUTLIER:
  MECHANIC:
COMPETITOR 2:
  OUTLIER:
  MECHANIC:
COMPETITOR 3:
  OUTLIER:
  MECHANIC:
```

## Wrap

- [ ] Move all captures into the idea inbox with a status of "Inbox."
- [ ] Don't develop any of them in this session. Triage is its own job.
- [ ] Mark the date so you can spot weeks you skipped.

```
TOTAL CAPTURES THIS WEEK:
STRONGEST 3 (gut pick, no committing):
  1.
  2.
  3.
```
